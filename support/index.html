Frequently Asked Questions

How do I convert multiple images at once?

To convert multiple images:

  1. Select multiple files using Cmd+Click or drag multiple files
  2. Click the "Batch Convert" button in the toolbar
  3. Choose your destination folder
  4. Monitor progress in the batch processing window

What image formats are supported?

We support conversion between the following formats:

  • PNG
  • JPEG
  • WEBP
  • AVIF
  • GIF
  • TIFF
  • SVG
